# Linked List Implementation in C

## Introduction
This project implements a **Linked List** data structure from scratch. It serves as a practice project to strengthen understanding of core data structures in computer science.

## Features
- Insert elements at head or tail
- Delete elements by value
- Search for elements
- Traverse and display the list
- Optional enhancements like reversing the list
- Reverse list
- Memory management

## Build
make

## Run
make run

## Output Example
10 -> 20 -> 30
10 -> 30
30 -> 10

## Concepts
- Pointer manipulation
- Dynamic memory allocation
- Data structure design
